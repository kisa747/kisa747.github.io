# XDG 基本目录

- `XDG_CONFIG_HOME`

  - 用于存放特定用户的配置（类似于`/etc`）。
  - 默认应为`$HOME/.config`。

- `XDG_CACHE_HOME`

  - 用于存放特定用户的非必要（缓存）数据（类似于`/var/cache`）。
  - 默认应为`$HOME/.cache`。

- `XDG_DATA_HOME`

  - 用于存放特定用户的数据文件（类似于`/usr/share`）。
  - 默认应为`$HOME/.local/share`。

- `XDG_STATE_HOME`

  - 用于存放特定用户的状态文件（类似于`/var/lib`）。
  - 默认应为`$HOME/.local/state`。
