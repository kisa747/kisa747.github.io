# Markdown 写作语法指南

## 基本语法

### 段落

Markdown标准语法：这是内嵌 `python` 代码， **这是加粗**，*这是斜体*，<u>这是下划线</u> ，这是链接：<https://docs.python.org>。

Markdown常用扩展语法：~~这是删除线~~ ，==高亮显示==，^^下划线^^ ，下标：H~2~O，上标：x^2^+y^2^=z，内嵌公式：$\Gamma(z) = \int_0^\infty t^{z-1}e^{-t}dt\,.$。
