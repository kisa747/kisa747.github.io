## Sandboxie-plus 笔记

Sandboxie-plus 是一款开源的沙盒隔离软件。专门用来治理哪些不得不用的、不守本分的国产软件。

>- 明明只需要用到核心的功能，却给我安装一堆功能无关的捆绑；
>- 即使退出了应用，依然有它的组件赖在后台不肯走；
>- 广告、资讯弹窗开机自启，屡禁不止；
>- 更有甚者，居然还会给系统安装驱动程序，而该驱动的作用与该应用的功能是无关的…
>- 比如：迅雷、夸克、百度云盘等。

便携化方法：

1. 下载 Sandboxie-plus，安装步骤中选择“提取全部文件到指定目录供便携化使用”，解压的目录中会有一个空白的 `Sandboxie.ini` 文件，此时的 Sandboxie-plus 已经是便携软件了，核心配置都在 `Sandboxie.ini` 文件中。
2. 更新 Sandboxie-plus 时，记得更新 `Sandboxie.ini` 配置文件，以免丢失配置信息。
3. 如果 `Sandboxie.ini` 配置文件不慎丢失，但已创建的沙箱目录文件还在，可以新建一个同名的沙箱，原来的沙箱就回来了。
