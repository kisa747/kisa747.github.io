# 添加远程打印机

## 远程添加家里的打印机

参考：<https://post.smzdm.com/p/awo5pwxp/>

1. 我需要的打印机不在列表中

1. 使用 IP 地址或主机名添加打印机，

   设备类型：TCP/IP 设备；

   主机名：`*.synology.me`，不需要带端口号

   不勾选“查询打印机并自动选择要使用的打印机驱动程序”

然后就会开始拼命的检测 TCP/IP 端口，会查很久，然后找不到。

3. 需要额外端口信息

​ 自定义 - 设置：协议选择`RAW`，修改端口号`20006`

4. 查找驱动，选择：hpygid37_v4.inf；选择 `HP Smart Tank 580-590 seriries PCL-3(V4)`
