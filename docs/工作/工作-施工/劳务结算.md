# 劳务结算

## 劳务结算

管井内阀门、管道、法兰、支架需要人工*1.3 系数

## 多结算的内容

1、给排水定额，P157，管道支架制作安装，室内管道公称直径 32mm 以下的安装工程已包括在内，不得另行计算。公称直径 32mm 以上的，可另行计算。

2、散热器上的手动放气阀。

3、7 号地的 PPR、PB 管结算，前期未扣除辅材价格。

首层排水管道超高费。

厨房、卫生间更换不锈钢篦子，空调套管加装护套，总结算时再对。
