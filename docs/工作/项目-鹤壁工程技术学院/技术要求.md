# 创优技术要求

## 烟感

参考《GB50166-2019 火灾自动报警系统施工及验收标准》，点型感烟火灾探测器应符合下列规定：

1. 探测器至墙壁、梁边的水平距离不应小于 0.5m；
2. 探测器周围水平距离 0.5m 内不应有遮挡物；
3. 探测器至空调送风口最近边的水平距离不应小于 1.5m，至多孔送风顶棚孔口的水平距离不应小于 0.5m；
4. 在宽度小于 3m 的内走道顶棚上安装探测器时、宜居中安装，点型感温火灾探测器的安装间距不应超过 15m，探测器至墙端的距离不应大于安装间距的一半；
5. 探测器宜水平安装，当确需倾斜安装时，倾斜角不应大于 45度。

## 喷头

参考《GB 50261-2017 自动喷水灭火系统施工及验收规范》

* 5.2.9 当梁、通风管道、排管、桥架宽度大于 1.2m 时，增设的喷头应安装在其腹面以下部位。

喷头与送风口国标无明确要求，但一般距空调送风口大于等于 30cm。

## 风管

参考：《GB 50243-2016 通风与空调工程施工质量验收规范》
6.3.1 悬吊的水平主、干风管直线长度大于 20m 时，应设置防晃支架或防止摆动的固定点。

风管末端应设置防晃支架。

## 压力表

压力表：

1. 安装要求：《GB 50093-2013 自动化仪表工程施工及质量验收规范》5.3.2，压力取源部件与温度取源部件在同一管段上时，应安装在温度取源部件的上游侧。

2. 量程选择：参考《16R405 暖通动力常用仪表安装图集》，被测压力比较稳定时，测量值不应超过量程的 2/3，测量值得最小值，不应低于量程的 1/3。压力仪表应安装在管道的直管段，并便于观察、维护。

## 温度计

1. 与管道呈倾斜角度安装时，宜逆着物料流向，取源部件轴线应与股电脑轴线相交；
2. 在管道的拐弯处安装时，宜逆着物料流向，取源部件轴线应与工艺管道轴线相重合。

## 水箱

参考图集《12S101 矩形给水箱》

1. 水箱透气管需安装防虫网
2. 水箱溢流管，水箱内做喇叭口，溢流管末端做防虫网
3. 水箱溢流管管径宜比进水管管径大一级；宜采用水平喇叭口集水；溢流管出口末端应设置用耐腐蚀材料制作的防护网；与排水系统不得直接连接并应有不小于 0.2m 的空气间隙。

参考《GB50015-2019 建筑给排水设计标准》

1. 生活饮用水水池（箱），进水管最低点高出溢流边缘的空气间隙不应小于进水管管径，且不应小于 25mm，可不大于 150mm。
2. 从生活饮用水管网向**消防**等其他非供生活饮用的贮水池（箱）补水时，其进水管口最低点高出溢流边缘的空气间隙不应小于 150mm。
3. 从生活饮用水管网向**中水、雨水回用水**等回用水系统的贮水池（箱）补水时，其进水管口最低点高出溢流边缘的空气间隙不应小于进水管管径的 2.5 倍，且不应小于 150mm。

## 末端试水装置

参考图集《20S206 自动喷水灭火设施安装》P65

1. 如果采用手动式，球阀（试水阀）安装在压力表之后，
2. 如果采用电动式，阀门（检修用）安装在电动试水装置之前
3. 试水阀和末端试水装置距地面的高度宜为 1.5m。

## 排水

* 伸顶通气管高出屋面（含隔热层）不得小于800mm。（规范中规定≥0.3m，且应大于最大积雪厚度）
